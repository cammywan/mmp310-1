JavaScript APIs

MMP 310 > Week 11

API stands for Application Programming Interface.

APIs are used to provide a set of protocols and patterns for building software and applications. Often they allow programmers to interact with a specific database, hardware or software.

We're going to look at a very simple API today, the Open Movie Database, a resource for movie information and media that is built on user contributions. The OMDb is a (mostly) free resource, and full access can be gained by donating money (this is just for poster images).

Simple OMBd synatax:

http://www.omdbapi.com/?t=Space Jam

Paste that into the browser window and you will see some simple JSON.

To use that in our own webpage, we can use a jQuery $.getJSON request:

$.getJSON('http://www.omdbapi.com/?t=Space Jam', function(json){  
  console.log(json);
});

We can use this information as part of our quiz show or another application.

$.getJSON('http://www.omdbapi.com/?t=Jurassic Park', function(json){  
  console.log(json);
});
