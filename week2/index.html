JavaScript Input and Variables

JavaScript is a scripting language, a lightweight programming language, specifically designed for programming the elements of web pages. Today we are going to look at some basic concepts in JavaScript and computer science in general which will allow us to interact with the browser and begin making dynamic designs. When we talk about dynamic design we mean making a design which changes given various inputs and interactions. The power of writing code is that we can build designs that react and change to various input, whether it is time, user interaction, data or other sources.

    With JavaScript, we can take inputs from the user and use them to modify our website or program in ways that we can't do with just HTML and CSS.

    Let's look at some JavaScript inputs.

    Button

    Button may the simplest form element:

    <button id="sumbit">Click me.</button>

    There are a lot of ways we can writes scripts to detect when the button is clicked. This is a type of event, and we'll talk more in depth about events next week. For now, let's write a very simple event handler to tell our program the button is clicked.

    var submit = document.getElementById("submit");
submit.onclick = function() {
	console.log(this);
};

    console.log

    console.log() is a special JavaScript function that can be used to report things that are happening in the program to the developer console. Like all languages, we need a way to print things to the console to see what works and doesn't it. It's extremely useful and easy to use, but you have to remember to be specific. You can put pretty much anything in the argument parentheses for console.log(), so make sure it's something useful, like an element, a variable, or something else that you want to be able to read. This is a good way to read data from various events.

    Input

    There are many different types of input. Today we'll just use a text input:

    <input id="name" type="text" placeholder="Input name" />

    We can use JavaScript to grad the value the user types in:

    submit.onclick = function() {
	var name = document.getElementById("name").value;
};

    A textarea is a resizable input for text.

    <textarea id="email" placeholder="Input email" />

    A select element creates a dropdown input using the option element.

    <select id="borough">
	<option>Bronx</option>
	<option>Brooklyn</option>
	<option>Manhattan</option>
	<option>Queens</option>
	<option>Staten Island</option>
</select>
