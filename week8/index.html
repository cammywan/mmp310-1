JavaScript time

MMP 310 > Week 8

There are cases in programming where you want something to take a certain amount of time to happen, or wait for a certain amount of time to execute.

In JavaScript, there are a few functions that can help us keep track of time.

The most simple example is the function Date();

Date();
Date.now();

JavaScript also has timers, including setTimeout and setInterval.

setTimeout delays the execution of a function for a specified amount of time measured in milliseconds.

var delayedGreeting = function() {
	alert("Hello world!");
};
setTimeout(delayedGreeting, 1000);

We can save a timeout as a variables and use that variable to cancel it.

var delayedGreeting = function() {
	alert("Hello world!");
};
var g = setTimeout(delayedGreeting, 1000);
clearTimeout(g);
